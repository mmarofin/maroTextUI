### How to use this TextUI

- just use this **INNOVATIVE** export in your ESX script

```lua
-- Standard (Green Success)
exports['maroTextUI']:TextUI("Press to open shop", "success", "E")

-- Info (Blue)
exports['maroTextUI']:TextUI("Press to view garage", "info", "G")

-- Error (Red)
exports['maroTextUI']:TextUI("You don't have enough money!", "error", "!")
```

- to hide it (forgot to mention)

```lua
exports['maroTextUI']:HideUI()
```
